# Data Sources — Módulo 1: Conceito Geral

#MachineLearning #DataSources #DataCollection

## O que são Data Sources?
- São as **origens dos dados** usados para treinar modelos de ML
- Sem dados de qualidade, não existe modelo funcional

## Princípio fundamental
> "Garbage in, garbage out" — dados ruins geram modelos ruins

## Tipos de Data Sources
- [[SQL Databases]] → dados estruturados e relacionais
- [[NoSQL Databases]] → dados flexíveis e não estruturados
- [[APIs]] → coleta de dados da internet em tempo real
- [[IoT]] → sensores e dispositivos físicos
- [[Data Formats]] → arquivos CSV, Excel, JSON, Parquet...

## Fluxo geral
Fonte de Dados → Coleta → Limpeza → Modelo ML

## Exemplos práticos
- Recomendação de filmes → [[SQL Databases]]
- Manutenção preditiva → [[IoT]]
- Detecção de fake news → [[APIs]] + datasets de texto
